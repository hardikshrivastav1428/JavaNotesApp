This project is a simple text-based Notes Manager built using Java File I/O.

## Features

- Add notes (append to file)
- View notes using FileReader + BufferedReader
- Clear notes (overwrite file)
- Uses try-with-resources and exception handling

## Concepts Used

- FileWriter (append & overwrite)
- FileReader
- BufferedReader
- Exception Handling (try-catch)
- try-with-resources

## How to Run

javac NotesApp.java
java NotesApp

## Output Example

1. Add Note
2. View Notes
3. Clear Notes
4. Exit
